Curso Eu Programo

Quando: 2016

O que: Identidade, ilustração, UX e front-end.

Dentre as ações feitas pela PrograMaria, causa da qual faço parte, foi criado o Curso Eu Programo, um curso de introdução à lógica de programação, HTML, CSS e JavaScript para mulheres.

A identidade foi pensada para representar o universo da programação de forma amigável – imenso, cheio de coisas interessantes para serem exploradas de forma divertida, leve.

Initial Ideas

Foram utilizadas analogias para traçar o paralelo com o universo – números binários, tags de html, propriedades css.
